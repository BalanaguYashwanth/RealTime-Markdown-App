# Real-time Markdown Backend

Backend was created using node.js with express

# Prerequisite	

| Prerequisite  | Version|
| ------------- | ------------- |
| Node  | v21.3.0  |
| NPM  | v10.2.4 |


# Installation Setup

```bash 
git clone https://github.com/BalanaguYashwanth/RealTime-Markdown-App.git
```

```bash 
cd Backend
```

```bash 
npm install
```

### Load .env

```bash 
copy contents from sample.env to .env
```
